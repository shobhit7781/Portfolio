# 🌐 Shobhit Arora — Personal Portfolio

A clean, animated personal portfolio website built from scratch with pure HTML, CSS, and JavaScript. Designed to showcase projects, skills, experience, and personality — with a polished UI, smooth scroll interactions, and a custom cursor effect.

🔴 **Live:** [your-portfolio-link.vercel.app]([Shobhit_Arora](https://shobhitarora.vercel.app/))  
📄 **Resume:** [View Resume](https://drive.google.com/file/d/11BVAWT3Tzd4HmV-UFhFLzpYIYlLRpn6n/view?usp=sharing)

---

## ✨ Features

- 🌀 **Animated loader** — ripple splash screen on every visit
- 🖱️ **Custom cursor glow** — soft radial glow that follows the mouse
- 🌊 **Parallax hero orb** — background orb that responds to mouse movement
- 📌 **Active nav highlight** — navbar links highlight based on scroll position
- 🎞️ **Scroll reveal animations** — sections fade and slide in on scroll using IntersectionObserver
- 📊 **Animated stat counters** — hero stats count up when they enter the viewport
- 🎫 **Infinite ticker strip** — scrolling marquee of quick facts and highlights
- 📱 **Fully responsive** — hamburger menu on mobile, desktop nav on larger screens
- 🔝 **Scroll-to-top button** — appears after scrolling down 400px
- 🧲 **Magnetic CTA buttons** — subtle hover interaction on hero buttons
- 🎨 **Glassmorphism cards** — frosted glass effect across all section cards

## 🛠️ Tech Stack

| HTML5 | Structure and semantic markup |

No frameworks. No npm. No build tools. Just files.
